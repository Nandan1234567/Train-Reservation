# Security Policy

## Supported Versions

Currently, only the latest version in the `main` branch is being actively supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues. 

Instead, please report them directly via email to **vadym.melezhyk.dev@gmail.com**. 

You should receive a response within 48 hours. If the issue is confirmed, a patch will be released as soon as possible, and you will be credited in the release notes.
